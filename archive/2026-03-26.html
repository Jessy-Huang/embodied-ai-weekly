精选论文 01 · cs.CV / cs.RO · 2026-03-25

从"抓错物体"到精准接触：TAG 用无目标引导机制根治 VLA 策略的实例级感知失灵

VLA 推理时干预 无需重训 LIBERO / VLABench
核心问题

中山大学团队通过大规模失败案例分析发现：VLA 策略在杂乱场景中的错误根源并非运动规划失败，而是实例级语义定位失败——策略在视觉遮挡或相似物体干扰下，产生"轨迹可信但目标对象错误"的幻觉式执行。

技术方案

提出 TAG（Target-Agnostic Guidance），借鉴图像生成领域的分类器自由引导（CFG）思想：同时计算①原始观测、②目标物体抹除观测下的策略输出，取其差作为残差引导信号强化物体证据权重。整个机制无需知道目标是什么（无目标先验），只需对比"有 vs 无物体"的策略差异。

⚡ 核心洞察：TAG 将 Diffusion/CFG 的引导范式迁移到 VLA 推理中，代价极低（无需训练）却效果稳健。这一范式提示：推理时的条件对比是提升模型鲁棒性的高效路径，值得在更多具身场景（如视频预测、物理推理）中探索。
🧩
TAG Architecture
CFG-inspired inference-time guidance for VLA robustness
Original obs ↔ Erased obs → Residual steering signal
TAG 方法架构示意：通过对比原始/抹除目标观测的策略差，构建无目标残差引导信号

精选论文 02 · cs.RO / cs.AI · 2026-03-25

机器人也有"情节记忆"：Chameleon 用几何锚定的多模态记忆栈攻克长视野操控的感知别名难题

情节记忆 长视野操控 感知别名 UR5e 真机验证
核心问题

操控任务中的感知别名（Perceptual Aliasing）：同一视觉观测可能对应不同操作历史，需要非马尔科夫决策。传统语义压缩式记忆会丢失关键消歧细节，相似度检索召回机制返回感知相似但决策无关的历史片段。

技术方案

提出 Chameleon：① 写入几何锚定的多模态 Token（3D 点云 + 语言特征联合嵌入），保留消歧所需精细感知线索；② 引入可微分记忆栈替代近邻检索，实现目标导向的定向回溯而非"最近邻"式记忆。配套发布 Camo-Dataset（UR5e 真机）。

⚡ 核心洞察：Chameleon 指向一个被长期忽视的问题——操控任务中"记什么"与"怎么查"同等重要。可微分记忆栈的设计使记忆检索本身可以端到端优化，这是对传统 episodic memory 架构的关键升级。

Geometry-grounded multimodal tokens + differentiable memory stack

精选论文 03-04 · 世界模型双连发 · 2026-03-25

隐空间世界模型的「双峰」爆发：DreamerAD 80x 提速 + Latent-WAM 104M 参数刷新 SOTA

世界模型 隐空间 RL NAVSIM v2 SOTA 端到端规划
DreamerAD（2603.24587）

首个基于隐空间世界模型的自驾 RL 框架，将 Diffusion 采样从 100 步压缩至 1 步（80x 加速）。三大机制：Shortcut Forcing（多分辨率递归压缩）、自回归稠密奖励模型（隐表示上直接操作）、Gaussian Vocabulary GRPO（约束探索于物理可行轨迹）。NavSim v2：87.7 EPDMS SOTA。

Latent-WAM（2603.24581）

仅 1.04 亿参数，在 NAVSIM v2 取得 89.3 EPDMS（超越前 SOTA 3.2 分），HUGSIM 28.9 HD-Score。核心：SCWE（空间感知压缩编码器，从基座模型蒸馏几何先验）+ DLWM（因果 Transformer 自回归预测未来世界状态）。

⚡ 具身迁移价值：这两篇工作的方法论对机器人操控"想象训练"高度适用——隐空间 RL 可绕过高成本真实机器人数据采集；SCWE 的几何蒸馏压缩范式可直接应用于操控场景的视觉编码器设计。小模型、高信息密度表示是下一代具身基座的核心路径。
